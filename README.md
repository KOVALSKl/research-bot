# research-bot

RAG-ассистент по научным исследованиям. Пользователи смогут задавать вопросы по PDF-исследованиям через бота ВКонтакте; поиск и генерация ответов выполняются отдельным API-сервисом.

## Текущий функционал (v1.1)

- Монорепозиторий с сервисом **core_api** (FastAPI) и общей библиотекой **research_shared**.
- Docker Compose с Qdrant для векторного хранилища.
- **Dense-эмбеддинги** через **Ollama** (`qwen3-embedding:0.6b`, multilingual).
- **Sparse-эмбеддинги** через fastembed (`Qdrant/bm25`) для терминного поиска.
- Гибридный поиск (семантический + по терминам) с fusion RRF.
- REST API: проверка состояния, добавление/удаление документов, поиск.

## Быстрый старт

```bash
cp .env.example .env
# Ollama должна быть запущена локально (порт 11434)
ollama pull qwen3-embedding:0.6b

docker compose up -d qdrant
uv sync
uv run core-api
```

API доступен на `http://localhost:8000`, документация — `/docs`.

> При смене dense-модели коллекция Qdrant пересоздаётся автоматически (dim 384→1024). Нужна переиндексация документов.

## Данные

PDF-исследования лежат в каталоге `researches/`.
